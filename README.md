# NRC-Cape-Generator-Improvend-expanded
This is the NRC Cape Generator 2.0
This is the NRC cape generator by einp2pe and Tomclapsyou. I've improved it in my opinion and added more features. I'm not entirely sure if everything works perfectly. Please let me know if there are any errors. With love, ~hykaika <3

Hex color code input

HSV color model for intuitive selection

Color preview and palettes

Drawing tools:

Various tools (brush, eraser, line, rectangle, circle)

Adjustable brush size and color

Layer system for front/back/elytra

Undo/Redo

Grid for better orientation

Improved UI:

Modern dark theme

Responsive design

Intuitive tab navigation

Visual feedback and animations

Enhanced state:

Auto-save to local storage

Comprehensive drawing data

Template support

Reset function

Canvas rendering:

Combination of all layers

Transparency support

Filters and effects
